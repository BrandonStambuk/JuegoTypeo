1.para poder usar las graficas se utiliza la siguiente libreria por lo que es necesario el uso de internet

https://cdnjs.cloudflare.com/ajax/libs/Chart.js/2.5.0/Chart.min.js

2. para poder iniciar el juego es necesario hacer un signUp
3. para ingresar al juego se realiza un signIn colocando sus credenciales del signUp
4. para iniciar el juego se presiona el boton empezar
5. para terminar se presiona el boton terminar
